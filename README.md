# Basic VM

## Softwares
* JAVA7
* Apache Tomcat 8
* VIM
* MySQL

## Configurations
* **OS:** Debian Jessie 64bits
* **IP:** 192.168.10.11
* **Shared folder:** ~/workspace/eng_soft/ => /srv/
* **RAM:** 512MB


## Prerequisites

### VirtualBox

### Vagrant

### Memory RAM

* 1024**MB**

## How to run?

1. Download this repository
2. Open your terminal
3. Run
	```
	$ vagrant up
	```
4. Enjoy
